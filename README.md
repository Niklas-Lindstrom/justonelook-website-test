A GitHub Pages website inspired by the Just One Look method. This is a test site for the Niklas-Lindstrom user.

## 📍 Live Website

Visit your site at: **https://Niklas-Lindstrom.github.io/justonelook-website-test/**

## 📁 Project Structure

```
├── index.html          # Main homepage
├── css/
│   └── style.css       # Styling and responsive design
├── js/
│   └── script.js       # Interactive functionality
└── README.md           # This file
```

## ✨ Features

- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Modern UI** - Clean and professional layout with gradient accents
- **Smooth Navigation** - Easy-to-use navigation with smooth scrolling
- **Sections Include:**
  - Hero section with call-to-action
  - The Method explanation
  - Testimonials
  - Featured articles
  - Newsletter signup
  - Contact form
  - Footer with links

## 🎨 Design Elements

- **Color Scheme:** Purple gradient (#667eea to #764ba2) with light backgrounds
- **Font:** Segoe UI, Tahoma, Geneva, Verdana, sans-serif
- **Layout:** Grid-based responsive design
- **Interactive Elements:** Hover effects, form submissions, smooth scrolling

## 🚀 Getting Started

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Niklas-Lindstrom/justonelook-website-test.git
   cd justonelook-website-test
   ```

2. **Open Locally:**
   - Open `index.html` in your web browser to preview

3. **Deploy to GitHub Pages:**
   - Your site automatically publishes to GitHub Pages when you push to the `main` branch
   - It's already configured in the repository settings

## 📝 Customization

### Edit Content
- **index.html** - Change text, add sections, modify HTML structure
- **css/style.css** - Customize colors, fonts, spacing, and responsive breakpoints
- **js/script.js** - Add or modify JavaScript functionality

### Common Customizations

**Change Colors:**
Find and replace the gradient colors in `css/style.css`:
- `#667eea` (primary color)
- `#764ba2` (secondary color)

**Update Text:**
Edit the content in `index.html` directly

**Modify Layout:**
Adjust CSS Grid and Flexbox properties in `css/style.css`

## 📱 Responsive Breakpoints

- **Desktop:** Full layout optimization
- **Tablet:** (max-width: 768px) - Adjusted padding and grid columns
- **Mobile:** (max-width: 480px) - Single column layout, larger touch targets

## 📧 Form Handling

Current forms show confirmation messages. To fully implement form submissions:
- Use a backend service (Firebase, Formspree, etc.)
- Or use a static form handler service
- Update the form submission handlers in `js/script.js`

## 🔗 Links & Navigation

All internal links use smooth scrolling. Update section IDs in HTML to match navigation href values.
